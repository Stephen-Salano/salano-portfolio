Do not add any text label next to or below the badge. The `S.S` characters are the entire logo. Nothing else.

## CHANGE 2 — CONSOLE EASTER EGG

Add a console easter egg that runs exactly once when the React application mounts. It must use `useEffect` with an empty dependency array in the root App component so it fires only on initial load, never on re-renders or navigation between pages.

Create a standalone utility function called `runConsoleEasterEgg` in a new file at `src/utils/consoleEasterEgg.js`. The App component imports and calls this function inside the useEffect. The function must not be called anywhere else.

The function uses `console.log` with the `%c` syntax to apply CSS styling to each line of console output. The browser's `%c` prefix allows CSS to be passed as a second argument to style the text — use this for every styled line.

The output renders in four distinct visual stages, each with its own colour and purpose:

Stage 1 — The Spring Boot ASCII banner. Render this in var(--green) (#22c55e) using font-family monospace and font-size 11px:

```
  .   ____          _            __ _ _
 /\\ / ___'_ __ _ _(_)_ __  __ _ \ \ \ \
( ( )\___ | '_ | '_| | '_ \/ _` | \ \ \ \
 \\/  ___)| |_)| | | | | || (_| |  ) ) ) )
  '  |____| .__|_| |_|_| |_\__, | / / / /
 =========|_|==============|___/=/_/_/_/
```

Stage 2 — The application name line. Render in var(--text-secondary) (#A9B7C6), monospace 11px:

```
 :: Stephen Salano Portfolio  :: (v1.0.0)
```

Stage 3 — The fake startup log lines. Render each line individually in var(--text-muted) (#6F737A), monospace 11px, with a small left padding for readability. The lines are:

```
[main] INFO  io.salano.portfolio.App      - Starting application...
[main] INFO  io.salano.portfolio.App      - JVM running for 0.842s
[main] INFO  io.salano.portfolio.Security - JWT filter chain initialised
[main] INFO  io.salano.portfolio.Config   - Multi-tenant context loaded
[main] INFO  io.salano.portfolio.Router   - Page routes registered: 5

✓ Application started in 1.337s
```

Note on the startup time: 1.337s is intentional. 1337 is leet speak and is a small joke visible only to developers who notice it. Do not change this number.

Stage 4 — The personal comment block. Render each line in var(--green) (#22c55e), monospace 11px. There must be one blank line before this block and one blank line after it:

```
// hey. you actually looked in here.
// most people don't.
//
// if you're reading this, you're probably the kind of
// engineer I'd want to work with.
//
// contact@stephensalano.dev
// github.com/Stephen-Salano
```

The complete function implementation looks like this. Copy it exactly:

```javascript
export const runConsoleEasterEgg = () => {
  const green = 'color: #22c55e; font-family: monospace; font-size: 11px;';
  const muted = 'color: #6F737A; font-family: monospace; font-size: 11px;';
  const secondary = 'color: #A9B7C6; font-family: monospace; font-size: 11px;';

  // Stage 1 — Spring Boot ASCII banner
  console.log(
    '%c  .   ____          _            __ _ _\n' +
    " /\\\\ / ___'_ __ _ _(_)_ __  __ _ \\ \\ \\ \\\n" +
    '( ( )\\___ | \'_ | \'_| | \'_ \\/ _` | \\ \\ \\ \\\n' +
    ' \\\\/  ___)| |_)| | | | | || (_| |  ) ) ) )\n' +
    "  '  |____| .__|_| |_|_| |_\\__, | / / / /\n" +
    ' =========|_|==============|___/=/_/_/_/',
    green
  );

  // Stage 2 — Application name
  console.log('%c\n :: Stephen Salano Portfolio  :: (v1.0.0)\n', secondary);

  // Stage 3 — Fake startup logs
  const logs = [
    '[main] INFO  io.salano.portfolio.App      - Starting application...',
    '[main] INFO  io.salano.portfolio.App      - JVM running for 0.842s',
    '[main] INFO  io.salano.portfolio.Security - JWT filter chain initialised',
    '[main] INFO  io.salano.portfolio.Config   - Multi-tenant context loaded',
    '[main] INFO  io.salano.portfolio.Router   - Page routes registered: 5',
    '',
    '✓ Application started in 1.337s',
  ];
  logs.forEach(line => console.log('%c' + line, muted));

  // Stage 4 — Personal comment block
  const message = [
    '',
    "// hey. you actually looked in here.",
    "// most people don't.",
    '//',
    "// if you're reading this, you're probably the kind of",
    "// engineer I'd want to work with.",
    '//',
    '// contact@stephensalano.dev',
    '// github.com/Stephen-Salano',
    '',
  ];
  message.forEach(line => console.log('%c' + line, green));
};
```

In the root App component, import the function and call it inside a useEffect:

```javascript
import { useEffect } from 'react';
import { runConsoleEasterEgg } from './utils/consoleEasterEgg';

// Inside the App component:
useEffect(() => {
  runConsoleEasterEgg();
}, []); // Empty array means this runs once only, on first mount
```
