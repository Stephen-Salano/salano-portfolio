# `<salano />`

> Personal portfolio for **Stephen Salano** — Full-Stack Engineer & CTO based in Nairobi, Kenya.

Live at: [stephensalano.dev](https://stephensalano.dev) *(in progress)*

---

## Overview

A developer portfolio built with a dark terminal aesthetic. The design language draws from developer comment syntax — `//` for section labels, `/* */` for callout blocks — with a consistent green-on-black colour system throughout. No stock illustrations. No generic UI kits. Just code, content, and intentional design.

---

## Pages

- **Home** — Full-scroll landing page covering the hero, featured work, experience timeline, technical stack, and a contact CTA.
- **Work** — Project cards for Psyllabus, File Flow API, and Courigistics Backend, plus an open source section linking to GitHub.
- **About** — Background, philosophy, and technical identity written in original voice.
- **Notes** — System design writeups and architecture notes.
- **Contact** — Inquiry form with an "Open To" list and a contact type selector.

---

## Design System

| Token | Value |
|---|---|
| Accent colour | `#22c55e` (green) |
| Background | `#0a0a0a` (near-black) |
| Font family | Monospace throughout |
| Logo | `<salano />` |
| Identity pill | `/* SaaS — Salano as a Service */` |

The hero section uses a CSS grid texture combined with a faint green radial glow. This same green fade background is repeated on every page's title section for visual consistency.
